计算机怎么把程序变成0 1

  1. 计算机先定好一些规则，比如每个语句都有两部分：操作符，内容。
  2. 操作符可能有赋值、声明、循环等，然后把把每个操作符编号，例0 赋值，1 循环，2 声明。
  3. var a = 1，可能变成2 'a',0 'a' 1，其中’a’也会变成0 1。
  4. 所有万物皆0 1
  5. 代码变成0 1的前提就是有很多规则。
